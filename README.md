# UI Каталог

Интерактивный каталог UI-компонентов для сборки сайтов под клиентов.

## Как использовать

1. Открой `index.html` в браузере
2. Выбери цвет, кнопки, шрифт, углы, анимацию
3. Нажми **Сохранить пресет** — получишь файл `preset.txt`
4. Отправь файл Диме — он сделает сайт в этом стиле

## Разделы

- 🎨 Цвета — 12 вариантов основного цвета
- 🔘 Кнопки — форма и стиль
- ✏️ Шрифты — шрифт заголовков
- 📐 Углы — степень округлости элементов
- ✨ Анимации — эффекты движения

## Деплой

GitHub Pages: включить в настройках репо → Pages → Branch: main
